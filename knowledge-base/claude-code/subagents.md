# Claude Code: субагенты

> Источник: code.claude.com/docs/en/sub-agents · сверено 2026-05-30.

Субагенты — специализированные помощники со своим контекстом, системным промптом, набором инструментов и правами. Claude делегирует задачу субагенту, когда она подходит под его `description`, а тот возвращает только итог (экономит контекст основной сессии).

## Где
- Проектные: `.claude/agents/*.md` (в git, для команды).
- Пользовательские: `~/.claude/agents/*.md` (все проекты). При совпадении имени приоритет у проектных.

## Формат файла
```markdown
---
name: code-reviewer
description: Ревью кода после изменений. Use proactively after writing significant code.
tools: Read, Grep, Glob, Bash      # опционально; по умолчанию наследует все
model: sonnet                       # опционально: sonnet | opus | haiku | inherit
---

Системный промпт субагента: кто он, что делает, как, чего избегать.
Чёткие критерии, пошаговый подход, формат результата.
```
- `name`, `description` — обязательны. `tools`, `model` — опционально.
- **`description` управляет авто-делегированием** — пиши ясно, когда субагента применять (фразы вроде «use proactively», «after X» помогают).
- Меньше инструментов = безопаснее и предсказуемее. Дешёвые задачи можно роутить на `haiku`.
