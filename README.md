# Aegis

**Aegis** — десктопное приложение под Windows 10/11 (C# / .NET 9 + **Avalonia**): оптимизатор, чистильщик, сканер безопасности и тюнер системы «для себя». Проверки разложены по вкладкам, каждый пункт объясняется простыми словами через «?»-подсказку, починка — по одному пункту или массово, с обязательным бэкапом перед правкой и откатом. Интерфейс — русский, рассчитан на человека, который совсем не разбирается в компьютерах.

> **Статус:** в активной разработке. Решение `Aegis.sln`: `Aegis.Core` (движок сканов и обратимых правок) · `Aegis.Scanners` (30 сканеров: 8 вкладок + раздел «Здоровье») · `Aegis.System` (Windows-слой: 30 пробников — WMI/реестр/процессы/SMART/датчики, обратимые правки, бэкап/откат) · `Aegis.Threats` (VirusTotal + ИИ-помощник Gemini→ChatGPT→Claude + веб-поиск) · `Aegis.App` (UI на Avalonia). **323 теста, сборка без предупреждений.** UI собирается и публикуется в Windows `.exe` даже на Linux (ADR 0006).

## Возможности

- **Сканы по вкладкам:** система, мусор, автозапуск, процессы, реестр, настройки/приватность, угрозы, драйверы, утилиты + раздел «Здоровье» (температуры, диски, память, батарея, вентиляторы).
- **Обратимость всего:** точка восстановления Windows и/или экспорт затрагиваемой ветки реестра **перед** правкой; удаление — в Корзину с предупреждением о безвозвратном; кнопка «Вернуть» — только там, где откат реально возможен.
- **Безопасность:** эвристический сканер угроз + сверка хэшей через VirusTotal (с троттлингом и кешем по хэшу) + локальный Защитник Windows; детектор майнеров.
- **ИИ-помощник:** объясняет находки простыми словами и ищет актуальные версии драйверов/утилиты через веб-поиск, ссылки — только на официальные сайты.
- **Здоровье:** реальные температуры/обороты/частоты через LibreHardwareMonitor, SMART дисков, стресс-тест под нагрузкой с авто-стопом по перегреву.

## Стек

- **C# 13 / .NET 9 + Avalonia 11**, MVVM (CommunityToolkit.Mvvm), DI, async/await.
- Доступ к системе: WMI (`System.Management`), реестр, службы, Process API, точечный P/Invoke, LibreHardwareMonitor.
- Логирование — Serilog. Тесты — xUnit. Публикация — single-file self-contained, win-x64.

## Сборка и запуск

```bash
dotnet build -c Release      # всё решение (собирается и на Linux)
dotnet test                  # все тесты (Core / Scanners / System / Threats)
```

Готовый `.exe` для Windows и параметры публикации (включая ключи ИИ/VirusTotal/поиска) — в [`docs/BUILD.md`](docs/BUILD.md). Запуск — от имени администратора (манифест `requireAdministrator`).
