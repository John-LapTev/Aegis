### Fixed
- **Программа не запускалась на Windows (первый живой прогон, 2026-06-24):** причина — `single-file`-сборка делалась без `IncludeNativeLibrariesForSelfExtract`, поэтому нативные библиотеки Avalonia (`libSkiaSharp`/`libHarfBuzzSharp`/`av_libglesv2`) лежали отдельными `.dll` рядом с `Aegis.exe`. При запуске одного `.exe` без них окно не рисовалось, и GUI-приложение (`WinExe`) **молча закрывалось** — без окна, ошибки и процесса в диспетчере (ровно симптом Ивана). Свойство `IncludeNativeLibrariesForSelfExtract=true` зашито в `Aegis.App.csproj` → теперь `.exe` действительно один файл (проверено публикацией: нативных `.dll` рядом нет, размер 41→97 МБ).
