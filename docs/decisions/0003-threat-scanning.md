# 0003. Поиск угроз (сканер безопасности)

- Статус: Accepted
- Дата: 2026-06-17

> ПЛАН. Кода ещё нет — фиксируем архитектурное решение до начала разработки.

## Контекст
Одна из ключевых задач Aegis — «**вычислить вредные процессы**»: поиск вирусов/троянов/
майнеров, подозрительных процессов, автозапуска, записей реестра. Нужно честно
определить **зону ответственности и ограничения** — мы небольшая команда, не
антивирусная лаборатория, и не можем сопровождать полноценную сигнатурную AV-базу.

Требования:
- Детект подозрительных процессов, автозапуска, реестровых записей, сетевых соединений.
- Детект **майнеров** (характерная нагрузка GPU/CPU + известные паттерны).
- Онлайн-сверка по репутации, насколько возможно без своей AV-инфраструктуры.

## Варианты
1. **Свой эвристический сканер + онлайн-сверка хэшей через VirusTotal API**,
   опционально базовая сверка с Windows Defender (`MpCmdRun`/AMSI). Реалистично для
   нашей команды: эвристика (процессы/автозапуск/реестр/сетевые соединения) + репутация
   файлов по 70+ движкам VirusTotal без поддержки своей базы. Майнеры — эвристика по
   нагрузке GPU/CPU + известные паттерны.
2. **Полноценный AV-движок с сигнатурами с нуля** — собственная база сигнатур,
   реал-тайм-защита, распаковка/эмуляция. **Нереалистично:** требует команды
   аналитиков, постоянного обновления базы, огромных ресурсов; гонка с авторами зловредов;
   высокий риск ложного чувства защищённости. **Отклонён.**
3. **Полностью офлайн-эвристика без онлайн-сверки** — проще, не зависит от сети, но
   без репутации хэшей резко выше доля ложных срабатываний и пропусков. Отклонён как
   единственный механизм; онлайн-сверка даёт качественный скачок точности.

## Решение
Строим **свой эвристический сканер** + **онлайн-сверку хэшей через VirusTotal API**,
**опционально** базовая сверка с Windows Defender (`MpCmdRun`/AMSI). Эвристика — по
процессам, автозапуску, реестру и сетевым соединениям; **майнеры** — по нагрузке
GPU/CPU + известные паттерны. **Полноценный AV-движок с сигнатурами с нуля НЕ делаем.**

**Ограничение фиксируем честно (в т.ч. в UI):** Aegis — вспомогательный сканер угроз
поверх штатной защиты Windows, а не замена антивирусу; реал-тайм-защиты и сигнатурной
базы нет; вердикты эвристичны и опираются на внешнюю репутацию (VirusTotal).

## Последствия
- **Плюсы:** реалистичный объём работ для команды; высокая точность по известным
  угрозам за счёт репутации VirusTotal; не вводим пользователя в заблуждение о
  «полноценном антивирусе».
- **Минусы / риски:** требуется **онлайн** для сверки хэшей; **VirusTotal API key**
  (хранить в `.personal`/env, не в репозиторий) с лимитами бесплатного тарифа (rate
  limits → батчинг/кэш хэшей); эвристика даёт ложные срабатывания — обязательны
  понятные «?»-Explain-тексты и обратимые действия; новые/целевые угрозы могут не
  определяться.
- **Требует от остального стека:** сканеры угроз как `IScanner`-плагины с находками
  `Finding` (severity + Explain) (Фаза 4); безопасное хранение API-ключа; любое
  действие по находке — через обратимую операцию (см. [0002](0002-backup-rollback.md),
  [0004](0004-elevation-and-safety.md)); логирование (Serilog).
