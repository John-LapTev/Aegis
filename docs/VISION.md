# Видение — Aegis

> Статус: **в разработке** (описано целевое видение продукта; каркас Фазы 1 + сканеры блока «А» уже созданы — см. [`ROADMAP.md`](ROADMAP.md)).

## Что строим
Aegis — десктоп-приложение (.exe) под Windows 10/11: оптимизатор + чистильщик + сканер безопасности + тюнер системы. Цель — «максимально правильно настроить компьютер и вычислить вредные процессы». Сканирует систему, раскладывает находки по вкладкам, объясняет каждую простыми словами и чинит обратимо (с бэкапом и откатом). UI — русский.

## Зачем / для кого
- **Для кого:** Иван (брат John). Приложение «для себя» — личный инструмент, без рынка и мультиязычия. UI только русский.
- **Задача:** Иван не разбирается в технологиях, но хочет, чтобы его ПК был чистым, быстрым и безопасным. Сейчас это значит вручную лезть в реестр, автозапуск, службы, искать вирусы и обновлять драйверы — слишком сложно и рискованно.
- **Ценность:** одно приложение, которое само находит проблемы, человеческим языком объясняет «что не так и что даст исправление», и чинит **безопасно и обратимо** — с бэкапом перед каждой правкой и автоматическим откатом, если ПК после починки перестал работать. Никаких безвозвратных удалений «на свой страх и риск».

## Группы функций (10 — со слов заказчика)
1. **Скан всей системы** — общий проход по компьютеру (Win 10/11).
2. **Драйверы — новые версии** — проверка, что установлены актуальные версии драйверов.
3. **Реестр** — поиск проблем и подозрительных записей.
4. **Автозапуск** — выявление подозрительных процессов в автостарте.
5. **Активные процессы** — поиск подозрительных/вредных среди запущенных.
6. **Системные настройки** — нарушения и неправильные настройки системы.
7. **Тюнинг GPU (NVIDIA/AMD)** — подбор оптимальных настроек драйвера под мощность ПК. Программа сама качает+ставит драйверы и применяет рекомендованные настройки **по нажатию кнопки**, с объяснением «почему так лучше» и через обратимую операцию (см. [ADR 0005](decisions/0005-drivers-and-gpu-tuning.md)).
8. **Мусор** — поиск и очистка ненужных файлов по всей системе.
9. **Угрозы** — поиск вирусов, троянов, майнеров.
10. **Недостающее** — выявление отсутствующих библиотек/драйверов под модель ПК.

## UX (ключевые принципы)
- **Результаты по вкладкам.** После скана находки разложены по группам (вкладкам), чтобы поэтапно проходить по найденному.
- **«?»-подсказки.** У каждого пункта — значок «?» в круге: при наведении короткий текст простыми словами — в чём суть проблемы и что именно даст её исправление и зачем.
- **Починка по одному или массово.** Можно исправить один пункт либо быстро выделить всё (массовое выделение) и исправить разом.
- **Бэкап ДО исправления.** Перед любой правкой создаётся точка восстановления / экспорт затрагиваемых веток. Если после починки ПК не загружается или глючит — при старте подхватывается бэкап (откат).
- **Flow перезагрузки.** Если починка требует ребута, после перезагрузки программа сама запускается с вопросом «всё ли работает корректно?»: «Да» → изменения остаются; «Нет, откатиться» → грузим бэкап.
- **Раздел «Бэкапы».** Отдельный экран: выбрать бэкап-файл/точку восстановления и восстановить вручную.
- **Доступность.** Высокий контраст на тёмном фоне, видимый фокус, крупные кликабельные зоны.

## Ключевые требования
- Платформа: только Windows 10/11, десктоп, самодостаточный .exe.
- Права администратора (requireAdministrator) — нужны для реестра, служб, драйверов, точек восстановления.
- **Обратимость любого изменения** — точка восстановления + бэкап ветки ПЕРЕД правкой; без безвозвратных удалений; явные предупреждения; полный лог операций.
- Онлайн нужен только для проверки новых версий драйверов и сверки угроз (VirusTotal); остальное работает офлайн.
- Дизайн — мягкий тёмный с градиентом, современный дашборд (см. [`DESIGN.md`](DESIGN.md) и [`DESIGN_BRIEF.md`](DESIGN_BRIEF.md)).

## Не входит в скоуп (пока)
- **Полноценный AV-движок** с сигнатурной базой с нуля — НЕ делаем. Поиск угроз — собственная эвристика + онлайн-сверка хэшей (VirusTotal), опционально базовая сверка с Windows Defender. Ограничение честно зафиксировано (см. ADR 0003).
- **Кастомный загрузчик / самописный bootloader** — НЕ делаем (риск «кирпича»). Откат при незагрузке — через штатное восстановление Windows (WinRE / точка восстановления). См. ADR 0002.
- **Мультиязычие** — UI только русский (продукт «для себя»).
- **Кросс-платформа** (macOS/Linux), мобильные клиенты, серверная/облачная часть.
