# Макеты экранов Aegis (утверждено)

> Утверждённый заказчиком (Иван) визуал — **референс** для реализации UI в Avalonia/XAML.
> Источник истины по токенам/правилам — [`../../DESIGN.md`](../../DESIGN.md); это его наглядное воплощение.

## Экраны
| Файл | Экран | Что показывает |
|------|-------|----------------|
| `results.png` · `screen-results.html` | Результаты по вкладкам | Главный рабочий экран: вкладки-группы, находки (чекбокс · статус · «?» · «Чинить»), раскрытый «?»-тултип, массовые действия. |
| `dashboard.png` · `screen-dashboard.html` | Дашборд / скан | Кольцо прогресса, счётчики, плашка железа, группы проверки. |
| `backups.png` · `screen-backups.html` | Бэкапы | Список точек/бэкапов (дата, что затронуто, размер, «Восстановить»), метка «авто». |
| `reboot.png` · `screen-reboot.html` | Диалог после ребута | «Всё работает корректно? Да / Откатиться» + авто-откат по таймауту. |

## Перерисовать
HTML — источник, PNG — рендер. Общие стили в `base.css`. Перерисовать (headless Chrome):

```bash
google-chrome --headless=new --no-sandbox --hide-scrollbars \
  --force-device-scale-factor=2 --window-size=1300,880 \
  --screenshot=results.png "file://$PWD/screen-results.html"
```

> Это статичные макеты для согласования облика, не код приложения. Реализация — в `src/Aegis.App/` (тема `Themes/Dark.xaml`, экраны `Views/`).
