# Правила проекта — индекс

Загружаются автоматически в каждой сессии (`.claude/rules/`).

- [`documentation-discipline.md`](documentation-discipline.md) — синхрон код/доки/структура/память.
- [`project-hygiene.md`](project-hygiene.md) — без мусора, чистый код.
- [`coding-standards.md`](coding-standards.md) — стандарты кода (C#/.NET 9 + Avalonia): MVVM/DI/async, обратимость, коллизия namespace, паттерн Windows-рантайма/тестируемости, путь «находка→починка».
- [`git-workflow.md`](git-workflow.md) — git, изоляция, коммиты по команде.
- [`release-process.md`](release-process.md) — выпуск версии: publish со всеми ключами + sha256, changelog, тег, GitHub Release (авто-обновление).
- [`ai-collaboration.md`](ai-collaboration.md) — порядок работы агента.
- [`dev-conduct.md`](dev-conduct.md) — код только в файлы, ответы коротко; сборка/тесты локально.
- [`session-log.md`](session-log.md) — непрерывность через `.session-log.md`.
- [`telegram-interaction.md`](telegram-interaction.md) — общение через Telegram: отвечать в том же канале; вопросы текстом (без интерактивных пикеров).
