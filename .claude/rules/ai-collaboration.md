# Правило: как агенту работать в этом проекте

- В начале сессии прочитай: CLAUDE.md → docs/VISION.md → docs/ROADMAP.md → docs/ARCHITECTURE.md.
- Для крупной задачи — сперва план, потом исполнение. Для исследований по многим файлам — субагенты/параллельные задачи.
- Решения, которые принимает только пользователь (продуктовые развилки, приоритеты, выбор стека) — спрашивай, не угадывай молча.
- Соблюдай `documentation-discipline` и `project-hygiene` в каждой задаче.
- С пользователем — по-русски, коротко и по делу.
