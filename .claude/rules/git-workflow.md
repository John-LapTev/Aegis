# Правило: git

## Изоляция
- Репозиторий **независимый**. Без общего git с соседними проектами в `/home/john-laptev/AI/`.
- Не добавляй remote на чужой репозиторий. Перед `push` проверяй `git remote -v`.

## Коммиты и push — только по явной команде
- **НИКОГДА** не `git commit` / `git push` без прямой просьбы пользователя. Перед коммитом покажи, что войдёт, и спроси.

## Сообщения
- Semantic commits на английском: `feat:`, `fix:`, `docs:`, `chore:`, `refactor:`, `test:`.

## Гигиена
- Не коммить `.personal/`, `.env`, секреты, `*.db`, `node_modules/`, артефакты.
